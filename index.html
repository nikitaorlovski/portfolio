⚡ Основные навыки

API & Валидация

  • Проектирование REST-эндпоинтов на FastAPI, роутинг, dependency injection.
  • Схемы данных и валидация через Pydantic, строгие аннотации типов.
  • Документация OpenAPI/Swagger, формирование примеров и описаний.

Данные и хранение

  • SQLAlchemy: декларативные модели, сессии, связи.
  • PostgreSQL: проектирование схем, индексы, SQL-запросы.
  • Redis (начальный уровень): кэширование и простые очереди.
  • Alembic: базовые миграции.

Инфраструктура

  • Docker: контейнеризация приложения и зависимостей.
  • Docker Compose (базово): запуск связанных сервисов (app, db, redis).
  • Git: ветки, pull-requests, базовый CI/CD (при необходимости).

Администрирование и тулзы

  • SQLAdmin: пробные интеграции админ-панели для моделей SQLAlchemy.
  • Линтинг/форматирование: ruff / black
  • Тестирование: pytest (базовое покрытие критичных частей).
